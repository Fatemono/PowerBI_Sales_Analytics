#  Проект: Анализ Розничных Продаж (Sales Analytics Dashboard)

##  Цель проекта
Демонстрация навыков моделирования данных (схема "Звезда"), написания DAX и визуализации ключевых метрик для Коммерческого Департамента.

##  Стек Технологий
* **Инструмент:** Microsoft Power BI Desktop
* **Языки:** Power Query (M), DAX
* **Концепции:** ETL, Схема "Звезда", Time Intelligence.

##  Ключевые Инсайты (Главные метрики)
* **Общая Выручка:** [Вставь сюда значение с дашборда]
* **Маржинальность:** [Вставь сюда значение с дашборда]
* **Динамика:** Виден [сезонный рост/спад] в [названия месяцев].

##  Основные этапы и демонстрация навыков
### 1. Моделирование данных
* Создана **Схема "Звезда"** с **Факт-таблицей** (`Факт_Продажи`) и **четырьмя Справочниками**: `Спр_Дата`, `Спр_Товары`, `Спр_Клиенты`, `Спр_География`.
* Все связи установлены как **"Один ко многим" (1:*)** и активны.

### 2. Разработка DAX-мер
Разработаны и протестированы ключевые меры:
* **Базовые:** `Общая Выручка`, `Общая Прибыль`.
* **KPI:** `Маржинальность %` (использована функция `DIVIDE`).
* **Анализ Времени (Time Intelligence):** `Выручка YoY` (использована функция `SAMEPERIODLASTYEAR` — ***Примечание: ввиду ограниченности данных, для визуализации использовалась симуляция `План Продаж`***).

### 3. Дизайн и Визуализация
* Создан дашборд, отвечающий на вопросы о **динамике**, **географии** и **маржинальности** товаров.
* Использованы **Условное форматирование** и **Срезы** для интерактивного анализа.
